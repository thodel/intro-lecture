It exists since the 1950s and the goal of computational linguistics is to understand
how humans use natural language to model the use of natural language, natural communication

So what is formal modeling and can you formally model linguistic data, language?
Yes you can.
There is work by Alexei Glattke and Igor Metschuk from the late 1960s in their book on elementary
mathematical linguistics in German translated as elementary mathematical linguistic.
And there they argue that to formally describe something and language in particular you have
to describe it in a way that the data is logically coherent, it's non-ambiguous and it's absolutely
explicit and then you can model it and ultimately you can build computer programs on it.
So this means that formal modeling of language of linguistic data means making explicit everything
that's the data itself, so the text or the relations within the text, so the code text
and also all the relations to the world around this text, this textual data is describing,
so the context.
And if you can do this then you can build a formal model and you also can run this formal
model in a computer program to test your theory behind your model building.
So digital linguistics in my understanding is the formal and thus computational modeling
of language data either spoken or written.
